    triple_barrier_labels(close, high, low, atr_window=20, upper_mult=2.0, lower_mult=1.0, horizon=5) -> pd.Series[int]: compute ATR(atr_window) via indicators.atr; for each bar i, upper = close[i] + upper_mult*ATR[i], lower = close[i] − lower_mult*ATR[i]; scan forward window close[i+1 : min(i+horizon+1, len)]; first touch of upper → +1, lower → −1, else 0 (time barrier). Guard NaN/zero ATR → 0. Use min(i+horizon+1, len(close)) to stay in bounds (Pitfall 4).

    trend_scan_labels(close, min_window=3, max_window=10, t_stat_threshold=1.5) -> pd.Series[int]: for each bar, over forward windows w in [min_window, max_window], fit scipy.stats.linregress(x, y); t-stat = result.slope / result.stderr (NOT rvalue — Anti-Pattern); track max |t-stat|; label = sign(slope) if max|t| >= threshold else 0.

    build_meta_features(df) -> pd.DataFrame: construct the exact 10 columns (ma_signal, macd_signal, rsi_signal, consensus_score, rsi_value, macd_hist, atr_norm, vol_pred, momentum_rank, onchain_health). Every derived column is shift(1) BEFORE return. consensus_score = (ma_s + macd_s + rsi_s)/3 on the shifted sub-scores. rsi_value via indicators.rsi(close).shift(1); macd_hist via indicators.macd(close)[2].shift(1); atr_norm = (atr(high,low,close)/close).shift(1). vol_pred/momentum_rank read from df columns (NaN fill if absent); onchain_health defaults to 0.5 when absent (D Feature Set). high/low fall back to close if missing.
